# Panadek PWA

A responsive Progressive Web App (PWA) built with React and TypeScript that delivers daily messages through an interactive scratch-off card interface. The application automatically shifts themes and content based on the user's local time of day.

## âœ¨ Features

* **Interactive Scratch Card:** Utilizes the HTML5 Canvas API to create a realistic, touch-responsive scratch-off effect with dynamic textures.
* **Time-Based Theming:** Automatically switches between "Morning" (Pink/Gold) and "Night" (Red/Dark) themes and message sets based on local time.
* **Progressive Web App (PWA):** Fully installable on mobile devices for a native app-like experience, complete with offline capabilities.
* **Responsive Design:** Built with Tailwind CSS to ensure a seamless experience across all device sizes.
* **TypeScript:** Strictly typed for robust and maintainable code.
